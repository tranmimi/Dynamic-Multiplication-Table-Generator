Create an Interactive Dynamic Table

Instructions:

Generate a multiplication table by filling out the fields below and clicking Submit. All inputs MUST be integers in the range [-50, 50]!

Valid inputs will produce a multiplication table. Each saved table will be stored in the Tabs section at the bottom of the page. Click on a tab with the desired inputs to revisit a multiplication table you created!

Click the "Save" button to save a table you created. Select one or more tables and click the "Delete" button to delete them.

Note: Larger ranges may take a while to load!
